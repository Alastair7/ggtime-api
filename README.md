# GGTIME API

## How to run
1. Clone the project.
2. `cd ggtime-api`
3. go run cmd/api-server/main.go

## How to run tests
### Run all tests
1. cd to project root `cd ggtime-api`
2. `go test ./...`

### Run specific test
1. `go test [folder-path]/`
2. `go test` to test only the files existing in the current directory.
